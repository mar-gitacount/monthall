# README

## 概要

このプロジェクトは、EVENCEのサイトからデータを抽出する `scriping.py` から構成されています。`scriping.py`のみで完結します。

## ファイル構成

- `scriping.py`: EVERNCEのサイトからデータを抽出する。


## 必要なライブラリ

以下のPythonライブラリが必要です。これらは `requirements.txt` ファイルに記載されています。

- requests
- BeautifulSoup4
- sqlite3
- pandas
- openpyxl

インストール方法:
```sh
pip install -r requirements.txt
